ChartView 组件布局设计

自定义 ViewGroup，可嵌入任何 XML 布局。所有子元素均为内部组件。

ChartView 组件结构（纵向 LinearLayout）
┌─ ChartView (LinearLayout vertical) ─────────────────────┐
│ ① 控制条 (ControlsRow)
├─ 图表样式 Spinner (折线图/平滑曲线/后置阶梯线)
├─ 时间范围 Spinner (全部/1天/1周/1月...)
└─ 全屏按钮 [⛶]
│ ② Y轴控制行 (YAxisRow)
├─ Y轴: [ 0 %] ~ [100 %]
│ ③ 图表内容区 (ChartCanvas) — 纯 Canvas 自绘
├─ 网格线和 Y轴刻度
├─ 数据曲线 (3种样式切换)
├─ X轴时间标签 (按时间范围间隔)
├─ 十字线 + 交点气泡
└─ [⛶] 全屏浮层按钮 (全屏时显示)
│ ④ 时间轴滑动区 (TimelineArea)
├─ ━━━━━■━━━━━ 进度条 + 位置块
└─ ◀ ████████████████ ▶ 交互拖拽区 + 左右按钮
│ ⑤ 图例行 (LegendRow)
├─ ━━━ 测量值
└─ --- 目标值
└──────────────────────────────────────────────┘

关键设计决策

纯 Canvas 自绘方案优势

  • 统一坐标系：图表数据、十字线、时间轴共用一套 Canvas 坐标
  • 无第三方依赖：去除 MPAndroidChart，减少约 2MB APK 体积
  • 完全控制：触摸事件、动画、样式全部可控
  • 更流畅：避免 View 覆盖层叠加导致的渲染开销

需要重建的功能

  • 数据曲线渲染（折线/贝塞尔/阶梯）
  • Y轴刻度计算和网格线
  • X轴标签定位和格式化
  • 触摸 → 数据点坐标映射
  • 十字线交点插值计算

图表内容区 Canvas 绘制层级

第1层 drawGrid() — Y轴水平网格线
第2层 drawDataLine() — 主数据曲线 + 数据点圆点
第3层 drawTargetLine() — 目标值虚线
第4层 drawXAxis() — X轴时间标签 + 刻度线
第5层 drawCrosshair() — 十字虚线 + 交点圆 + 信息气泡

坐标系统设计（统一使用相对时间戳）

时间轴 (X轴)
chartMinX = 0 (最左 = 可见窗口最早时间点)
chartMaxX = visibleRange (最右 = 可见窗口最晚时间点)
数据点 X = dp.timestamp - windowStartTimestamp

数值轴 (Y轴)
chartMinY = 用户设定最小值 (百分比换算)
chartMaxY = 用户设定最大值 (百分比换算)

屏幕像素 = (值 - 轴最小值) / (轴范围) * 图表区像素
